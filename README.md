# Branch Practice

## 1차 목표
- 3개의 브랜치에서 만든 기능 merge
```
1. Main 에서 login, score, ranking 브랜치를 생성
2. 기능을 각 브랜치에서 구현
    - login 브랜치
        회원가입, 로그인, score 저장, 전체 테이블 전달 기능
    - score 브랜치
        1ms마다 1씩 score가 오르는 기능
    - ranking 브랜치
        순위 정렬 후, 3위까지 표시
3. login, ranking Merge 이후 main에 Pull Request
4. main 기반 score에서 작업 후, main에 Pull Request
```
